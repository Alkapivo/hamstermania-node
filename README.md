# HamstermaniaNode

## About

Simple nodejs server, build using [patchwire](https://github.com/twisterghost/patchwire).


## Instruction 

Install:
```
npm install
```

Run:
```
npm start
```
## Client

Compiled clients can be found in `release` folder. 

Available versions:

| Platform | Version |
|----------|---------|
| Windows  | 0.4.3   |
| Ubuntu   | 0.4.3   |
| Android  | 0.4.3   |
